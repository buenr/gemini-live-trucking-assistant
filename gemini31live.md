4. Function Calling (Tool Use) Example (Python)
Gemini 3.1 Flash Live supports synchronous function calling to enable the model to request external tool functions during a conversation. You can declare available functions in the session config, detect tool invocation events, and respond with results: [ai.google.dev], [ai.google.dev]
Pythonimport wavefrom google import genai, genai, typesclient = genai.Client()model = "gemini-3.1-flash-live-preview"# Define available tool (function) declarationsturn_on_the_lights = {"name": "turn_on_the_lights"}turn_off_the_lights = {"name": "turn_off_the_lights"}tools = [{"function_declarations": [turn_on_the_lights, turn_off_the_lights]}]config = { "response_modalities": ["AUDIO"], "tools": tools }async def main():    async with client.aio.live.connect(model=model, config=config) as session:        prompt = "Turn on the lights please."        await session.send_client_content(turns={"parts": [{"text": prompt}]})                # Prepare to save or play the model's audio output        wf = wave.open("response_audio.wav", "wb")        wf.setnchannels(1); wf.setsampwidth(2); wf.setframerate(24000)  # 24kHz mono PCM                async for response in session.receive():            if response.data is not None:                # Audio chunk from model’s response                wf.writeframes(response.data)            elif response.tool_call:                print("Tool call requested:", response.tool_call.function_calls)                # For each requested function, prepare a dummy success result                function_responses = []                for fc in response.tool_call.function_calls:                    function_responses.append(types.FunctionResponse(                        id=fc.id,                        name=fc.name,                        response={ "result": "ok" }                    ))                # Send function results back to the model                await session.send_tool_response(function_responses=function_responses)        wf.close()asyncio.run(main())Show more lines
Sources: Official docs (function calling usage in Live API) [ai.google.dev], [ai.google.dev]
